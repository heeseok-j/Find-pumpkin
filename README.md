## 3. 利用方法
 スタートボタン押すとゲームが始まり、 パンプキンを探してクリックする</br>
 停止ボタン押すとアイコンが消え、残り時間がとまり、その逆の場合はアイコンが現れ時間が流れる</br>
 パンプキンを全部探して勝ちにするか、時間切れや他のアイコンを押してしまって負けになるかのゲームです
